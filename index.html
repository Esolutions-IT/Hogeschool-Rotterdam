ONS PROJECT

SMART object

Voor de nieuwe sprint moeten wij een smart pushy robot gaan maken. Met alle kennis die we opgebouwd hebben moet dit wel lukken. Voor een smart object heb je ook sensoren nodig en om hem een stevige casing te geven gebruiken wij de lasersnijder.

Sensoren

Voor een smart object heb je ook sensoren nodig. Wij gebruiken de ultrasoon bewegings sensor. Hierdoor kunnen wij meten wat de afstand tussen de andere robot is en daar dus de wapens op af te richten.

Casing

Om de robot stevig te maken hebben wij een casing gebouwd. Dit hebben wij gedaan bij de lasersnijder. Alle draden en hardware kan in die casing.
